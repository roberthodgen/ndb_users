JSON API

GET _login.json
Determines if a user is logged in, fetches information about the logged in user, or logs out the current user.
POST _login.json
Logs in a user.
POST _login/create.json
Creates a new user.
GET _login/activate.json
Activates a user's account (verify email address).
POST _login/password/change.json
Changes the logged in user's password.
POST _login/password/forgot.json
Requests a password recovery token via email.
GET _login/password/reset.json
Verifies a password recovery token is usable.
POST _login/password/reset.json
Resets a user's password using a password recovery token.
